Fetch data from Athena tables.

Using the library pyathena we set the user path to an s3 bucket. Then we pass the desired query returning a dataset.

Parameters:

  • quert (string) – An SQL query

Returns:

df – A pandas dataframe containing the results of the query

dataframe

read_from_athena(‘s3://jonathan-oak/s3’, ‘’’select * from “fda-open-database”.”recall” ‘’’)
